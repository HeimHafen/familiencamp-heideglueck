# Familiencamp Heideglück (Demo)

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Online-1f7a3a)](https://heimhafen.github.io/familiencamp-heideglueck/)
![Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS-lightgrey)
![Status](https://img.shields.io/badge/Status-Demo%20Showcase-brightgreen)

Dieses Projekt ist eine fiktive Demo-Website für einen familienfreundlichen Campingplatz.  
Ziel ist zu zeigen, wie ein moderner Auftritt für kleinere Plätze aussehen kann – mit klarer Struktur und mobilem Fokus.

- **Live:** https://heimhafen.github.io/familiencamp-heideglueck/
- **Tech-Stack:** HTML, CSS, leichtes JavaScript – statisch ausgeliefert über GitHub Pages.
- **Inhalte:** Startseite, Plätze & Lodges, Umgebung, FAQ, Kontaktanfrage – aus Sicht von Familien gedacht.
- **Zweck:** Referenzprojekt im Portfolio von HeimHafen Web, um Stil und Struktur für Camping- und Tourismusprojekte zu demonstrieren.

## Seitenstruktur

- Übersichtlicher Hero mit klarem Nutzenversprechen
- Präsentation von Stellplätzen und Unterkünften
- Abschnitt „Umgebung & Ausflüge“
- Häufige Fragen (FAQ)
- Kontakt / unverbindliche Anfrage

Das Design ist bewusst ruhig gehalten, orientiert sich an kleinen, familienfreundlichen Plätzen und ist vollständig mobil optimiert.
