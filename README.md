# Familiencamp Heideglück (Demo)

Dieses Projekt ist eine fiktive Demo-Website für einen familienfreundlichen Campingplatz.  
Ziel ist zu zeigen, wie ein moderner Auftritt für kleinere Plätze aussehen kann – mit klarer Struktur und mobilem Fokus.

- Live: https://heimhafen.github.io/familiencamp-heideglueck/
- Tech-Stack: HTML, CSS, ein wenig JavaScript – kein Build-Setup, statisch über GitHub Pages.
- Inhalte: Startseite, Plätze & Lodges, Umgebung, FAQ, Kontaktanfrage – aus Sicht von Familien gedacht.
- Zweck: Referenzprojekt im Portfolio von HeimHafen Web, um den Stil für Camping- und Tourismusprojekte zu demonstrieren.
